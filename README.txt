CAUTION: This is NOT an official release of MultiWii. 

It is a stripped down version of version 2.2 intended for Atmel 32u4 based boards using 
an MPU6050 examples of which are NanoWii and the HK Pocket Quad.

The implementation is limited to 4 motors which may be brushed or brushless and no servos.

Baro, Mag, Sonar and GPS etc are not supported. Baro may be added later.

There are major code rewrites from the original 2.2. The minimum cycle time is around 1.4mS.
